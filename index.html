OSN-HAX - Unofficial Extensions to the Oracle Social Network Web Client

Installation

  • If you are using Firefox, install Greasemonkey from here.
  • If you are using Chrome, install Tampermonkey from here.
  • Once you have those extensions installed, click the button below.
  • Once the script is installed, navigate back to your OSN window, and refresh.
  • You will find the Extensions settings under the usual OSN Settings menu.
  • Further instructions on using the extensions are on their way.
